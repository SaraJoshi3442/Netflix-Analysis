# 🎬 Netflix Content Analysis using Python

## 📌 Project Overview

This project explores a Netflix dataset using Python to uncover trends in content distribution, genres, countries, IMDb ratings, and popularity scores through Exploratory Data Analysis (EDA).

The goal of this project was to practice data cleaning, data manipulation, and visualization while answering real-world business questions using Pandas, Matplotlib, and Seaborn.

---

## 📂 Dataset

The dataset contains information about Netflix Movies and TV Shows, including:

* Title
* Type (Movie / TV Show)
* Genre
* Country
* Release Year
* IMDb Score
* Popularity Score
* Runtime
* Director
* Cast
* Netflix Original

---

## 🛠️ Technologies Used

* Python
* Pandas

* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Business Questions Answered

* Are there more Movies or TV Shows on Netflix?
* Which genres are the most popular?
* Which genres have the highest IMDb ratings and popularity scores?
* Which countries produce the most Netflix content?
* Which countries produce the most Movies and TV Shows separately?
* What are the Top 10 Most Popular Netflix Titles?

## 💡 Key Insights

* Netflix offers both Movies and TV Shows with a diverse global catalog.
* Drama, Comedy, and Action dominate the platform.
* Some genres achieve high IMDb ratings despite having fewer titles.
* A small number of countries contribute the majority of Netflix content.
* Popularity and IMDb score do not always move together, highlighting differences between audience engagement and critical ratings.
